# d3js

## 簡介

D3.js（D3或Data-Driven Documents）是一個使用動態圖形(svg)進行資料視覺化的JavaScript程式庫。與其他的程式庫相比，D3對檢視結果有很大的可控性。

簡單的圖片辨識指南：&#x20;

* JPG：用在尺寸較大的照片，以減少檔案大小。&#x20;
* PNG：有效減少圖片檔案大小以及保留照片透明元素。&#x20;
* SVG：向量檔，縮放不失真。&#x20;
* GIF：可以製作小型動態圖片檔案。
